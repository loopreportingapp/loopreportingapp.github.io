Application Information

The Graphical Reporting Application was developed by Sylver Yagi (Website Designer/Engineer) of PNG LOOP. The purpose of designing and building this application is to make reporting interesting with the help of graphs.

It was built using Open Source Resources namely Bootstrap 3, Font Awesome Icons and D3.js. The main CSS base of the application is Bootstrap 3.0. A mixture of a number of icons are used including Font Awesome Icons and Bootstrap's Glyphicons.

D3.js is a JavaScript Library used to display data stored in a json (JavaScript Object Notation) file.

Right now it is limited to only showing the Reporter Story count. In the second updated version it will have the capabilities to show the count for stringers also.
